## Answer Frameworks

### Technical Q&A (2–3 minutes)

```
1. Direct answer (15 seconds)
2. How it works (45 seconds)
3. Trade-offs / when NOT to use (30 seconds)
4. Production example (30 seconds)
```

### System Design (45 minutes)

```
1. Requirements clarification     (5 min)
2. Capacity estimation            (5 min)
3. High-level design              (10 min)
4. Deep dive — 2 components       (15 min)
5. Bottlenecks & scale            (5 min)
6. Failure modes & monitoring     (5 min)
```

### Behavioral — STAR Method

```
Situation  → Business context (30 sec)
Task       → Your responsibility (15 sec)
Action     → What YOU did, specifically (60 sec)
Result     → Measurable outcome (15 sec)
```
