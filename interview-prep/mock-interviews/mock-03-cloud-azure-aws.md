# Mock Interview 03 — Cloud Architecture (Azure + AWS)

> **Duration:** 60 minutes | **Weeks 46 prep

## Format

| Block | Time | Topic |
|-------|------|-------|
| Azure | 25 min | WAF, landing zones, compute choice |
| AWS comparison | 20 min | Map same workload to AWS |
| Multi-cloud | 10 min | Portable vs lock-in |
| Q&A | 5 min | |

## Interviewer Questions

### Azure (from [Azure Top 50](../azure-top-50-index.md))
1. "Design subscriptions for 5 product teams with SOC 2."
2. "When App Service vs AKS vs Functions for .NET order API?"
3. "How do you achieve 99.95% SLA multi-region on Azure?"

### AWS (from [AWS Top 50](../aws-top-50-index.md))
4. "Map the same order API to AWS — draw services."
5. "IAM roles vs users — how does CI/CD deploy?"

### Multi-cloud
6. "CTO wants cloud-agnostic. What's actually portable?"

**Look for:** FARCS framework, service mapping tables, honest lock-in assessment.

## Scoring

| 5 | Maps services correctly, trade-offs explicit, production examples |
| 3 | Knows services but weak on governance/scale |
| 1 | Name-dropping without requirements |
