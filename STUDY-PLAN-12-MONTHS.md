## Weekly Schedule Template


## Prerequisites Checklist

Before Week 1, confirm:

- [ ] 5+ years professional .NET/C# development
- [ ] Built and shipped at least one production web application
- [ ] Basic SQL knowledge (SELECT, JOIN, indexes)
- [ ] Familiarity with REST APIs and HTTP
- [ ] Azure or AWS account (free tier sufficient for labs)
- [ ] Docker Desktop installed
- [ ] Git proficiency
- [ ] 10+ hours/week available for 52 weeks
