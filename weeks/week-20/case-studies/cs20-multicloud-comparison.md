# Case Study: Azure vs AWS for .NET SaaS

## Context
B2B SaaS, 5K tenants, .NET 8, team knows Azure, board asks AWS evaluation.

## Deliverables
1. Architecture diagram per cloud
2. Cost estimate (12-month)
3. Migration effort estimate
4. Recommendation with ADR

## Reference
| Factor | Lean Azure | Lean AWS |
|--------|-----------|----------|
| Team skills | .NET + Entra | Need training |
| Existing EA | Microsoft EA | None |
| AI features | Azure OpenAI native | Bedrock |
| Market perception | Enterprise Microsoft shops | Broader SaaS ecosystem |

**Recommendation for .NET shop:** Azure primary. AWS literacy for interviews and multi-cloud optionality.
