## Q071: Portfolio Executive Summary

| Attribute | Value |
|-----------|-------|
| **Difficulty** | Advanced |
| **Category** | Graduation Capstone |
| **Frequency** | Very Common |

### Question

How write one-page portfolio executive summary for principal architect search?

### Short Answer (30 seconds)

Specialty, 3 impact metrics, target role, proof links — scannable in 60 seconds.

### Detailed Answer (3–5 minutes)

**Structure:**
1. Headline specialty (Azure platform, regulated fintech)
2. Three bullets: $ saved, uptime improved, teams scaled
3. What you seek (principal IC, platform org)
4. Links: PDF, LinkedIn, top talk

**Tone:** Confident not arrogant; metrics verified.

### Architecture Perspective

Executive summary is recruiter funnel — optimize scannability.

### Follow-up Questions

1. **Photo? — Optional on PDF; professional if used.**
2. **Objective statement? — Replace with specialty + seek.**

### Common Mistakes in Interviews

- Wall of buzzwords
- No metrics
- Generic 'senior architect'

## Q072: Portfolio Diagram Standards

| Attribute | Value |
|-----------|-------|
| **Difficulty** | Advanced |
| **Category** | Graduation Capstone |
| **Frequency** | Common |

### Question

What diagram standards for portfolio case studies?

### Short Answer (30 seconds)

C4 L1–L2 minimum; legend; consistent notation; before/after optional; no live confidential URLs.

### Detailed Answer (3–5 minutes)

**Standards:**
- C4 context + container per case
- Consistent colors/shapes
- Legend on each diagram
- Data flow labels
- Anonymized names (Client A, Payment Service)

**Tools:** Structurizr export, Mermaid, draw.io — pick one style.

**Anti-pattern:** 40-box unreadable diagram.

### Architecture Perspective

Diagram standards signal professionalism — messy diagrams undermine senior claim.

### Follow-up Questions

1. **Sequence diagrams? — One per case for critical flow.**
2. **ADR excerpts? — 1 page max per case.**

### Common Mistakes in Interviews

- Screenshot of prod console
- No legend
- Inconsistent notation across cases
