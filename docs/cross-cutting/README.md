## How to Study Cross-Cutting Topics

1. **First encounter:** Read overview when referenced from a weekly module
2. **Deep dive:** Complete the dedicated module before Month 11
3. **Interview prep:** Review all cross-cutting topics in Weeks 45–47
4. **Case studies:** Apply cross-cutting lenses during Week 44 marathon

## Interview Q&A Target: 200+ across all cross-cutting topics
