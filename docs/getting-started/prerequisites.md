# Prerequisites

Complete this checklist before starting Week 1.

## Experience Requirements

| Requirement | Minimum | Your Status |
|-------------|---------|-------------|
| Professional software development | 5+ years | ⬜ |
| .NET / C# production experience | 3+ years | ⬜ |
| Built and deployed a web application | Yes | ⬜ |
| Worked with relational databases | Yes | ⬜ |
| Used Git in a team environment | Yes | ⬜ |
| Consumed REST APIs | Yes | ⬜ |

**Ideal candidate:** 10–15 years .NET development, team lead or senior engineer role, limited formal architecture exposure.

## Technical Skills (Baseline)

You should be comfortable with (not expert in):

- [ ] C# syntax, LINQ, async/await
- [ ] ASP.NET Core MVC or Web API
- [ ] Entity Framework Core basics
- [ ] SQL: SELECT, JOIN, INSERT, UPDATE, basic indexes
- [ ] HTTP, REST, JSON
- [ ] Git: branch, merge, pull request workflow
- [ ] Command line basics (cd, ls, grep)

You do **not** need prior experience with:
- Kubernetes, Terraform, system design interviews
- AWS or Azure (we teach from fundamentals)
- Design patterns beyond basic Singleton/Factory awareness
