## How to Work Through Case Studies

1. Read business context only — don't peek at reference solution
2. Set a 60-minute timer
3. Produce: architecture diagram + written decision rationale
4. Compare with reference solution
5. Score yourself using the assessment rubric
6. Discuss with study group if available
